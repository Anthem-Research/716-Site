# 716 Site

Static website for 716.

## Structure

index.html contains the page content.

styles.css contains the visual system and responsive layout.

script.js controls the fixed menu interaction.

.github/workflows/pages.yml deploys the site to GitHub Pages.

## GitHub Pages setup

The repository includes a GitHub Actions workflow for Pages deployment.

In GitHub, open Settings, then Pages, then Build and deployment. Set Source to GitHub Actions.

After that, every push to main will deploy the site.

Expected public URL format:

https://anthem-research.github.io/716-Site/

## Editing

Replace the placeholder project blocks in index.html with real project images, captions, and case study links as the body of work develops.
